Medical Appointment No-Show - Data Cleaning & Preprocessing

Project Overview
This project focuses on cleaning and preprocessing the Medical Appointment No-Show dataset obtained from Kaggle. The objective is to improve data quality by handling missing values, correcting data types, removing invalid records, and preparing the dataset for Exploratory Data Analysis (EDA) and Machine Learning.
Dataset
Source: Kaggle
Dataset: Medical Appointment No-Show Dataset
Description: The dataset contains information about medical appointments and whether patients attended or missed their scheduled appointments.

Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy

Data Cleaning & Preprocessing Performed
The following data preprocessing steps were carried out:
- Imported and explored the dataset
- Checked dataset shape and column information
- Identified missing values
- Checked duplicate records
- No missing or duplicate records
- Validated Age column and removed invalid age values
- Converted date columns into appropriate datetime format
- Renamed columns for better readability
- Replaced categorical values where required
- Checked unique values in categorical columns
- Verified data types of all columns
- Exported the cleaned dataset for further analysis
